🚀 启动说明

要启动开发服务器，请在终端中运行：

cd matrix-website
npm run dev

然后访问 http://localhost:5173/pindou-generator

📋 使用流程

  1. 上传图片 (JPG/PNG/WEBP)
  2. 调整设置 (尺寸、难度等)
  3. 使用工具编辑拼豆图案
  4. 选择颜色进行手动调整
  5. 导出 PDF 图纸或 PNG 图片
