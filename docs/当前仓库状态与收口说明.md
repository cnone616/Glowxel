# 当前仓库状态与收口说明

更新日期：2026-03-26

## 一、仓库概况

当前仓库包含 5 个主要端：

- `website/`：官网与在线创作站
- `admin/`：后台管理
- `server/`：后端 API
- `uniapp/`：客户端与设备控制
- `esp32-firmware/`：ESP32 固件
