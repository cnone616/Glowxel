# 项目架构与接口文档

更新日期：2026-04-24

说明：本文默认站在仓库实现层描述架构，出现 `Glowxel` 时指仓库 / 工程代号；对外产品名分别是 `光格像素工坊` 与 `Glowxel PixelBoard`。

## 一、模块结构

仓库当前分为 5 个主模块：

- `website/`
  - 官网、社区、在线创作、图纸工作台、基础网页设备入口
- `admin/`
  - 运营后台
- `server/`
  - API 服务
- `uniapp/`
  - 小程序 / H5 / App 客户端，承接主设备控制链
- `esp32-firmware/`
  - ESP32 设备固件与设备网页

补充入口：

- 页面分类总入口见 [文档与页面索引.md](/Users/aflylong/Desktop/project/Glowxel/docs/文档与页面索引.md)
- 当前开发主线见 [开发路线图.md](/Users/aflylong/Desktop/project/Glowxel/docs/开发路线图.md)
- `website` 路由入口见 `website/src/router/index.js`
- `admin` 路由入口见 `admin/src/router/index.js`
- `uniapp` 页面目录位于 `uniapp/pages/`

## 二、当前职责边界

### `website`

负责：

- 官网浏览壳
- 应用主端壳
- 社区浏览
- Web 创作链
- 图纸工作台
- Web 基础设备控制
- Web 热点配网引导
- Web 设备参数页
- Web 画板模式
- 工作台与公开用户页

当前不负责：

- `uniapp` 全量设备模式等价
- 时钟、宠物、屏保等完整设备模式页
- 微信网页授权 UI
- 手机号登录字段合同之外的猜测认证能力
- `weather / countdown / stopwatch / notification` 网站正式入口

### `admin`

负责：

- 用户、作品、评论、模板、挑战、活动管理
- 固件版本管理
- 基础运营数据看板

### `uniapp`

负责：

- 像素创作
- 社区互动
- 热点配网引导
- 主设备控制链
- 时钟编辑
- 主题模式
- 宠物眼睛模式
- 与设备实时同步
- 微信登录后的账号绑定链

### `server`

负责：

- 用户与管理员鉴权
- 项目与作品数据
- 社区互动数据
- 固件版本接口
- 文件上传

### `esp32-firmware`

负责：

- 设备配网
- 设备网页
- WebSocket 命令接收
- 像素画布显示
- 静态 / 动态时钟显示
- 主题渲染
- 原生业务效果
- 动画播放
- OTA 更新

### 用户

前缀：`/api/user`

主要接口：

- `POST /admin-login`
- `POST /login`
- `GET /profile`
- `PUT /profile`
- `GET /stats`
- `GET /recommended`
- `GET /search`
- `GET /settings`
- `PUT /settings`
- `GET /:id`
- `GET /:id/artworks`

当前认证口径：

| 项目 | 当前状态 |
| --- | --- |
| `POST /api/user/login` | 保留为 `uniapp` 微信登录正式入口 |
| `POST /api/user/admin-login` | 继续作为仓库现有 Web 账号密码登录入口；若后续切普通用户手机号登录，必须先冻结字段映射表 |
| 网站本地会话 | `auth_token` |
| 网站当前用户缓存 | `currentUser` |
| 网站登录时间 | `loginTime` |
| 微信身份字段 | `openid` |
| 微信联合标识 | `union_id` |
| 手机号注册/登录/绑定/合并字段 | 尚未冻结，必须先补 `field -> source` 映射表，再改代码 |

## 四、设备通信

### HTTP

设备 HTTP 接口由固件侧提供，当前主链仍包含：

- `/status`
- `/test`
- `/upload`
- `/brightness`
- `/text`
- `/clear-wifi`
- `/themes`
- `/effects`
- `/theme-frame.bin`
- `/live-frame.bin`
- `/theme-frame.txt`
- `/live-frame.txt`

### WebSocket

设备 WebSocket 路径：

- `ws://<device-ip>/ws`

当前设备主模式：

- `clock`
- `animation`
- `canvas`
- `theme`

当前网站正式使用的设备状态字段：

- `ip`
- `width`
- `height`
- `brightness`
- `mode`
- `businessMode`
- `effectMode`

当前业务模式标签还包括：

- `gif_player`
- `led_matrix_showcase`
- `tetris`
- `tetris_clock`
- `game_screensaver`
- `planet_screensaver`
- `eyes`

当前网站模式可见性：

- 正式设备模式：`canvas`
- 正式设备链页面：`/device-control`、`/ble-config`、`/device-params`、`/canvas-editor`
- 桌面创作强入口：`/workspace`、`/create`、`/editor/:id?`、`/overview/:id`、`/assist/:id`、`/publish-project/:id`、`/pattern-workbench`
- 社区与账号链：`/community`、`/artwork/:id`、`/user/:id`、`/profile`、`/my-works`、`/my-favorites`、`/achievements`、`/cloud-sync`、`/settings`
- 本轮排除：`weather / countdown / stopwatch / notification`

当前网站路由分组：

- 官网浏览壳：`/`、`/community`、`/templates`、`/challenges`、`/challenge/:id`、`/artwork/:id`、`/user/:id`
- 应用主端壳：`/workspace`、`/create`、`/pattern-workbench`、`/editor/:id?`、`/overview/:id`、`/assist/:id`、`/publish-project/:id`、`/profile`、`/my-works`、`/my-favorites`、`/achievements`、`/cloud-sync`、`/settings`、`/settings/profile`、`/followers`、`/following`、`/device-control`、`/ble-config`、`/device-params`、`/canvas-editor`
- 公开集合页：`/gallery`
- 隐藏 QA 路由：`/design-preview`、`/design-compare`

当前命令已按类别拆分：

- 基础信息
- 模式切换
- 时钟配置
- 主题配置
- 动画传输
- 画布与像素显示
- 文本与原生效果
- 眼睛交互
- OTA 相关
