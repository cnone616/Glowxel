# Glowxel Website

官网与在线创作站点，基于 Vue 3 + Vite。

## 当前功能

- 首页、社区、挑战、模板、个人主页
- 在线创作页与像素编辑页
- 图纸工作台：`/pattern-workbench`
- 网页设备控制页：`/device-control`
- Web 时钟编辑页：`/clock-editor`
- 作品详情、挑战详情
- 登录与基础个人资料编辑
- 静态硬件指南页：`public/hardware-guide.html`

## 开发命令

```bash
npm install
npm run dev
npm run build
npm run preview
```
