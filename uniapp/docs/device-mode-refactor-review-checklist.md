1. 连接设备，打开控制页。
2. 点“静态时钟/动态时钟”按钮，不进入编辑。

预期：
- [x] 设备端模式不变化（代码链路完成，联调由你侧执行）。

### 用例 B：时钟编辑提交才切设备
步骤：
1. 从控制页进入时钟编辑（静态）。
2. 修改任一参数并点击发送。

预期：
- [x] 先切换设备到 `clock`，再应用配置（代码链路完成，联调由你侧执行）。

### 用例 C：拼豆退出自动恢复
步骤：
1. 当前模式设为 `animation`（已应用）。
2. 进入拼豆页（canvas）。
3. 返回上一页。

预期：
- [x] 进入拼豆时设备为 `canvas`（代码链路完成，联调由你侧执行）。

### 用例 D：异常断连恢复

1. 在拼豆模式中断开网络或关闭小程序。

预期：
- [x] 固件不会长期停留在 `canvas`（代码链路完成，联调由你侧执行）。

3. 点击“保存并应用”。
