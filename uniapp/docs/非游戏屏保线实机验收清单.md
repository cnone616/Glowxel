# 非游戏屏保线实机验收清单（2026-04-14）

适用范围（不含 `game_screensaver`）：

- `control`
- `ambient-editor`
- `led-matrix`
- `gif-player`

## 0. 验收前置

1. 设备已连接，控制页显示在线状态。
2. 画布预览均显示 `64×64` 网格线。
3. 当前固件与前端版本已更新到本轮分支。

## 1. 控制页入口（`control`）

操作步骤：

1. 进入控制页，确认“模式入口”为单一入口视图。
2. 点击各入口（除 `game_screensaver`）能进入对应页面。
3. 返回控制页后再次点击任意入口，确认文字不变大、不换行、不抖动。

通过判定：

- 不存在“主迁移入口 / 全部入口”切换。
- 入口可达，返回后入口样式稳定。

失败判定：

- 仍出现双入口分流切换。
- 点击入口后文字大小、字重或换行突变。


操作步骤：

2. 调整关键参数后点击发送。
3. 返回控制页再进入，确认参数保持。

通过判定：

- 预览帧持续播放，发送成功。
- 设备显示与预览语义一致。

失败判定：

- 预览不动或明显失真。
- 发送后设备表现与预览明显不一致。

## 3. 看板组

用户已最终确认：`天气看板 / 倒数看板 / 计时看板 / 提醒看板` 不适合当前 `64×64` 屏幕观感，相关 uniapp 页面已直接删除。

本轮不再对这四页做实机验收。

## 4. 场景与离线播放器（`ambient-editor` + `led-matrix` + `gif-player`）

操作步骤：

1. `ambient-editor`：切换至少 3 个 preset（含 `rain_scene`）并发送。
2. `led-matrix`：切换至少 3 个 scene 并发送。
3. `gif-player`：切换至少 2 个离线素材并发送。
4. 三页底部 tab 切换时观察文本与图标尺寸是否稳定。

通过判定：

- 每页预览帧数 > 0，播放连续。
- 发送命令成功且设备端可见对应效果。
- 选中态仅颜色/边框变化，不出现尺寸跳变。

失败判定：

- 选中态导致文字或图标变大。
- 发送后设备无变化或与预览严重不符。

## 5. 协议一致性抽检

抽检命令：

- `set_ambient_effect`
- `set_gif_animation`

通过判定：

- 字段与 `docs/非外接依赖全量参数映射表.md` 一致。
- 非 GIF 页面不调用 `uploadAnimationFrames`。

失败判定：

- 出现未约定字段、别名字段、兜底字段。
- 非 GIF 页面出现帧上传链路。
