# Glowxel 64x64 像素展示系统 V1 任务清单

## 1. 文档用途

本文档用于将《[pixel-display-system-planning.md](/Users/aflylong/Desktop/project/Glowxel/uniapp/docs/pixel-display-system-planning.md)》中的规划拆解为可追踪的执行任务。

使用方式：

1. 每完成一项任务，直接勾选对应条目
2. 若任务范围变化，在原任务下补充说明，不另起一套口径
3. 若任务涉及接口或协议变更，必须先补参数映射表，再进入编码

## 2. 当前执行原则

V1 只聚焦以下主线：

1. 建立轻量主题系统
2. 打通主题与素材引用
3. 保持静态时钟 / 动态时钟 / 画板 / 效果页现有链路不被破坏
4. 不扩展到通知、智能家居联动、消息卡片
5. 不做重资源整包主题系统

## 3. 任务状态说明

1. `[ ]` 未开始
2. `[-]` 进行中
3. `[x]` 已完成
4. `阻塞` 代表依赖未满足，不能继续推进

## 4. 里程碑

### 里程碑 A：主题系统边界确定

目标：

1. 定清主题、素材、算法效果三层边界
2. 不进入重资源主题包路线

### 里程碑 B：主题系统前端落地

目标：

1. 时钟编辑页支持主题选择
2. 第一批主题预设可切换可预览

### 里程碑 C：素材引用打通

目标：

1. 已保存素材可被主题或动态时钟引用
2. 页面预览和板载播放链路打通

### 里程碑 D：动画能力优化

目标：

1. 评估并提升长素材的播放能力
2. 处理帧数、存储、内存约束
