# 闹钟自定义功能说明

## 功能概述

用户现在可以自定义 LED 矩阵的闹钟显示样式，包括时间、日期、星期的位置、大小和颜色。

## 使用流程

### 1. 进入闹钟编辑器

1. 打开小程序/App
2. 进入"设备控制"页面
3. 连接到 ESP32 设备
4. 切换到"闹钟模式"
5. 点击"自定义闹钟样式"按钮

### 2. 自定义设置

#### 时间显示
- **字体大小**：1-3 级（1=小，2=中，3=大）
- **X 位置**：0-64 像素（水平位置）
- **Y 位置**：0-64 像素（垂直位置）
- **颜色**：9 种预设颜色可选

#### 日期显示
- **开关**：可以隐藏日期
- **字体大小**：1-2 级
- **X/Y 位置**：0-64 像素
- **颜色**：9 种预设颜色可选

#### 星期显示
- **开关**：可以隐藏星期
- **X/Y 位置**：0-64 像素
- **颜色**：9 种预设颜色可选

### 3. 实时预览

编辑器顶部有一个 64x64 的预览画布，所有修改都会实时显示效果。

### 4. 保存和应用

- **保存按钮**（💾）：保存配置到本地，下次打开编辑器时会自动加载
- **发送按钮**（📤）：将配置发送到 ESP32 设备，立即生效

## 技术实现

### uni-app 端

1. **闹钟编辑器页面**：`pages/clock-editor/clock-editor.vue`
   - Canvas 实时预览
   - 滑块控制位置和大小
   - 颜色选择器
   - 本地存储配置

2. **WebSocket 通信**：`utils/webSocket.js`
   - 新增 `setClockConfig()` 方法
   - 发送 JSON 格式的配置数据

### ESP32 端

1. **配置结构**：`ClockConfig` 结构体
   - 时间、日期、星期的独立配置
   - 支持字体大小、位置、RGB 颜色

2. **持久化存储**：使用 `Preferences` 库
   - 配置保存在 Flash 中
   - 断电后配置不丢失

3. **WebSocket 命令**：`set_clock_config`
   - 接收配置 JSON
   - 立即应用并保存
   - 如果在闹钟模式，立即刷新显示

## 默认配置

```cpp
ClockConfig clockConfig = {
  .time = {
    fontSize: 1,
    x: 17,
    y: 18,
    r: 100, g: 200, b: 255  // 青色
  },
  .date = {
    show: true,
    fontSize: 1,
    x: 3,
    y: 30,
    r: 120, g: 120, b: 120  // 灰色
  },
  .week = {
    show: true,
    x: 23,
    y: 44,
    r: 100, g: 100, b: 100  // 深灰
  }
};
```

## 注意事项

1. **位置范围**：确保文字不会超出 64x64 的显示范围
2. **字体大小**：较大的字体需要更多空间，注意调整位置
3. **颜色对比**：选择对比度高的颜色以便清晰显示
4. **设备连接**：必须先连接到 ESP32 才能发送配置
