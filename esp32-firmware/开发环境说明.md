# ESP32 开发环境说明

## 环境配置

### 已安装工具
- **Python3**: 已安装并配置
- **PlatformIO**: 已安装并可用
- **命令**: 使用 `platformio` 或 `pio` 命令

## 烧录流程

### 编译和烧录命令
```bash
cd esp32-firmware
python3 -m platformio run --target upload
```

### 常见问题

#### 端口被占用
如果遇到 "port is busy" 错误，需要先关闭占用串口的进程：

```bash
# 查找占用串口的进程
lsof | grep cu.usbserial

# 关闭进程（替换 PID 为实际进程号）
kill <PID>

# 等待2秒后重新烧录
sleep 2 && python3 -m platformio run --target upload
```

### 重要提示
1. **烧录完成后无需额外操作** - 不要执行串口监视命令
2. **环境已就绪** - 无需每次检查 Python 和 PlatformIO 是否安装
3. **直接执行烧录** - 可以直接运行 platformio 命令

## 串口监视（手动执行）

如需查看串口输出，手动运行：
```bash
python3 -m platformio device monitor
```

注意：此命令会持续运行，按 Ctrl+C 退出
