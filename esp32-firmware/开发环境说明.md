# ESP32 Glowxel PixelBoard 开发环境说明

更新日期：2026-04-07

说明：本文里的固件实现仍属于 `Glowxel` 仓库；对外设备正式名称统一为 `Glowxel PixelBoard`。

## 1. 环境配置

### 已安装工具

- Python 3
- PlatformIO

推荐命令：

```bash
cd esp32-firmware
platformio run
platformio run --target upload ### 烧录
platformio device monitor ### 串口打印
```

### 常见问题

- 端口被占用：`lsof | grep cu.usbserial`
- 无法识别串口：检查数据线和 `ls /dev/cu.usb*`
- 编译异常：优先重新执行 `platformio run`，不要手改 `.pio/` 中间产物
